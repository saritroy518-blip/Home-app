Data Imports

Manual import only — no API connections required. Export files from each system, import here. The dashboard auto-detects common column formats and populates the matching fields.

PioneerRx Import

Build a custom report in PioneerRx and export as CSV. Column mapper below handles any layout.

Recommended custom report columns: Rx Status (or Claim Status), Price (or Charge Amount), Drug Cost (or Ingredient Cost), Patient ID (or Patient Name), Prescriber Name, Fill Date, Rx Number. Include these and the importer auto-fills Scripts Filled, Reversed, Abandoned, Revenue, COGS, Active Patients, and Active Doctors.

Sage Intacct Import

Export from Sage Intacct: Reports → Financial → select report → Export → CSV. Supports P&L (Income Statement), Balance Sheet, and AP Aging reports. Maps account lines to dashboard fields automatically.

Bank Statement Import

Export your bank statement as CSV, select your bank format below, then import. The importer reads the ending balance and applies it to the Cash field.

Supported formats: Flagstar (Date, Description, Amount, Balance) · Chase (Date, Description, Amount, Balance) · Bank of America (Posted Date, Payee, Amount, Running Bal.) · Wells Fargo (Date, Amount, *, *, Description). Select your bank before importing.

Power BI / Dashboard Backup Import

Import a previously exported Power BI JSON file back into the dashboard. Restores all daily history, channels, and locations — useful for moving data between devices or browsers.

Power BI Export

Export all dashboard metrics in a structured format for Power BI. Includes daily history, channel breakdown, and location breakdown as separate tables.

Power BI setup: Power BI Desktop → Get Data → JSON (or Text/CSV) → Connect to exported file → Load → The dataset has three tables: daily_metrics, channels, and locations. Build visuals from there. Re-export and refresh on your schedule (daily or weekly).

How to Use

Daily: Enter numbers in Daily Numbers tab. Red/yellow alerts on Scorecard are your management agenda. Each team tab shows team-specific KPIs and OKRs.

Imports: Go to Imports tab → choose your system → export file from that system → import here. PioneerRx auto-fills Ops metrics. Intacct auto-fills financial fields. Bank auto-fills Cash. All file-based, no API keys.

Power BI: Export → Power BI Export (JSON) from Imports tab. In Power BI Desktop: Get Data → JSON → load file → three tables (daily_metrics, channels, locations). Refresh by re-exporting and re-loading on your cadence.

Weekly: Review Channels and Locations tabs. Kill bad volume. Fix the biggest leak. Export CSV for records.

Data storage: Saves to browser localStorage automatically as you type (debounced). Use Export Backup to protect data or move between computers. Import Backup to restore.
